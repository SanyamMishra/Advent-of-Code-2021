
# Advent of Code 2021

Solutions for the Advent of Code 2021


## Run Locally

Clone the project

```bash
  git clone https://github.com/SanyamMishra/advent-of-code-2021.git
```

Go to the project directory

```bash
  cd advent-of-code-2021
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
